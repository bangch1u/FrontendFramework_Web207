Những kỹ năng bạn sẽ được trang bị

Bạn sẽ được trang bị những kỹ năng thiết yếu nhất để trở thành một Data Analyst, bao gồm SQL, Python và Data Visualization từ cơ bản đến nâng cao. Từ đó bạn có thể xử lý và phân tích dữ liệu trong thực tế một cách hiệu quả.

Chương 1: Ứng dụng SQL
Nội dung:

SQL

Thực hành:

Nâng cảo khả năng sử dụng SQL, từ đó kết nối với BigQuery và Google cloud-based data ware house để truy vấn dữ liệu nhanh chóng

Chương 2: Lập trình cơ bản với Python
Nội dung:

Python

Thực hành:

Làm việc với string bằng Regex. Trích xuất dữ liệu từ website một cách hiệu quả

Chương 3: Làm quen với Pandas
Nội dung:

Pandas

Thực hành:

Nắm vững những bước cơ bản của quá trình chuẩn bị dữ liệu và khai thác insight một cách hiệu quả bằng Python’s Pandas library

Chương 4: Phân tích dữ liệu nâng cao
Nội dung:

Matplotlib, Seaborn, Tableau

Thực hành:

Phát triển các kỹ năng trình bày để biến insight thành các phương án xử lý bằng cách học cách tạo ra các hình ảnh trực quan có tác động

Chương 5: Machine Learning
Nội dung:

Machine learning cơ bản

Thực hành:

Sử dụng các thuật toán Machine Learning để rút ra những quy luật và dự đoán từ bộ dữ liệu
